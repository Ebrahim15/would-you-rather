# would-you-rather
 "Would-You-Rather" Game

## Description

A simple "Would You Rather" game.


### Built with

- HTML, CSS
- ReactJS, Redux

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
